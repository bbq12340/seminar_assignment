# 파이썬 과제

### 2023-23441 장재혁

Code

```python
age = int(input('당신의 나이를 입력하시오:'))
if age <= 10:
    print('10대 이하입니다.')
elif age <= 20:
    print('10대 입니다.')
elif age <= 30:
    print('20대 입니다.')
elif age <= 40:
    print('30대 입니다.')
elif age <= 50:
    print('40대 입니다.')
else:
    print('50대 이상입니다.')
```

Output

```
당신의 나이를 입력하시오:26
20대 입니다.
당신의 나이를 입력하시오:90
50대 이상입니다.
```
